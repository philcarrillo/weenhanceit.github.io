Web Applications

Why Use a Web Application

A Web Application (often called a web app) uses a standard web browser, such as Chrome, Firefox or Internet Explorer, to implement a customer software application. Since pretty well all computers are running browsers, and most users are comfortable using browsers - implementing a Web Application avoids the hassles associated with distributing and maintaining custom application to each user. Web Applications are available to any user, any where in the world, as long as they are connected to the web.

Web Applications are implemented within the Internet Server that it supplying content to the user's browser. This means that the application logic is coded in one place - new releases are instantly available to all users with no action on their part. Properly designed applications can be accessed with all popular browsers, and there is a further advantage in that the application can be accessed by users on completely different operating systems.

Web Apps are particularly suited to database applications. In these applications all users have access to the latest dataset and problems associated with data synchronization can be greatly simplified.
